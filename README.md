# 📚 API REST - Livraria

API REST desenvolvida com **Node.js**, **Express** e **MongoDB** como parte dos meus estudos em desenvolvimento back-end.

O projeto tem como objetivo simular o gerenciamento de uma livraria, aplicando conceitos importantes de construção de APIs REST, integração com banco de dados e boas práticas de desenvolvimento.

> ⚠️ Projeto em desenvolvimento.

---

## 🚀 Tecnologias utilizadas

- JavaScript
- Node.js
- Express
- MongoDB
- Mongoose

---

## ✨ Funcionalidades

Atualmente a API conta com funcionalidades como:

- Cadastro de livros
- Consulta de livros
- Atualização de informações
- Remoção de registros
- Busca de livros
- Filtros
- Paginação
- Tratamento de erros

Novas funcionalidades serão adicionadas conforme o avanço dos estudos.

## ▶️ Como executar o projeto

1. Clone este repositório

```bash
git clone https://github.com/SEU-USUARIO/api-livraria-node.git
```

2. Acesse a pasta do projeto

```bash
cd api-livraria-node
```

3. Instale as dependências

```bash
npm install
```

4. Configure as variáveis de ambiente.

5. Execute a aplicação

```bash
npm run dev
```
